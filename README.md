# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built with **React** and **Vite**, showcasing projects, skills, and experience in Robotics, AI, and Full Stack Development.

## ✨ Features

- **Fast & Lightweight** — Powered by Vite for instant HMR and optimized builds
- **Responsive Design** — Looks great on all devices (mobile, tablet, desktop)
- **Project Showcase** — Highlights key projects with descriptions and tech stacks
- **Skills Section** — Visual display of technical skills and tools
- **Contact Section** — Easy ways to get in touch
- **Smooth Animations** — Clean UI transitions and interactions

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/PaarthSuwi/Portofolio_web.git
cd Portofolio_web

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
